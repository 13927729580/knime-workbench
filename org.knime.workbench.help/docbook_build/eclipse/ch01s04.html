4. Starting the Application

In the KNIME root directory (knime_1.1.2) the application is started via

  • knime.exe(for Windows) and

  • knime (Linux).

After the application has been started the KNIME application window appears (the screen shots in the following haven been taken from the Windows version).

KNIME is implemented as an Eclipse Plugin. For details on using the Eclipse workbench we refer to the User Guide at http://www.eclipse.org/documentation/main.html.

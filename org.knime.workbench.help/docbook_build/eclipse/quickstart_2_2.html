3.2. Linux users

Download the standalone version of KNIME from http://www.knime.org/download/linux/knime_linux_1.1.2.tar.gz. Extract the downloaded .tar.gz file knime_linux_1.1.2.tar.gz, to any directory for which you have write permissions.

This is all that is needed to install KNIME. There is no need to execute a Setup application or set any variables. KNIME will not leave many traces in your system. It will create a .knime sub-directory in your home directory to store its local settings.

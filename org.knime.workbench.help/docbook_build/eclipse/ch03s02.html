2. Requirements

Linux operating system with Java 5.0 RTE (or any later version) installed. If Java is not installed on your system, the standalone version of KNIME contains a Java runtime version that will be used automatically. If you have an earlier version of Java installed and you intend to install the KNIME-plugins directly into your own eclipse installation, you need to update from this website: http://java.sun.com/j2se/1.5.0/download.jsp
